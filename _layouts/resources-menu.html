OpenGL:

LearnOpenGL (Great tutorial series.)
OGLDev Modern OpenGL Tutorials (Another great tutorial series, gets a little more advanced.)
opengl-tutorial Tutorial 3 : Matricies (Particularly good section on matrices. Rest of the tutorial is great too.)

Socket Programming:

Beej's Guide to Network Programming (The best socket programming guide, informative and short. HTML version.)
Beej's Guide to Network Programming (PDF version.)
BSD Sockets: A Quick And Dirty Primer (Great short snippet to get straight into sockets.)
Computerworld Article on Sockets (State of socket programming in todays day and age. Interviews with Beej and Jim Frost.)
UNIX Network Programming by Richard Stevens. (Probably the best and most in depth book on everything networking. PDF version.)
Microsoft | Getting Started with Winsock. (Good for windows specific differences in socket programming.)
Serialization in Object-Oriented Programming Languages by Konrad Grochowski, Michal Breiter and Robert Nowak. (Network programming involes a lot of serialisation and this explains it pretty well!)
Gaffer On Games (Great articles on heaps of different networking topics.)
Multiplayer Doom on Cloudflare Workers | (Proof that emscripten can work with sockets.)
1500 Archers on a 28.8: Network Programming in Age of Empires and Beyond | (Article on networking problems faced in Age of Empires.)
